# Dự án: Med-Translator

## 1. Tên dự án & Mục tiêu cốt lõi
- **Tên:** Med-Translator
- **Mục tiêu:** Xây dựng một hệ thống dịch thuật tài liệu chuyên ngành y tế bằng trí tuệ nhân tạo (Gemini AI). Dự án giúp người dùng tải lên các tài liệu (thường là PDF), hệ thống tự động xử lý, chia nhỏ, dịch thuật và tổng hợp kết quả chính xác.

## 2. Đối tượng sử dụng
- **Người dùng cuối (End-user):** Sinh viên y khoa, bác sĩ, hoặc người nghiên cứu cần dịch nhanh tài liệu y khoa từ ngoại ngữ sang tiếng Việt.

## 3. Các tính năng chính
- **Tải lên & Quản lý file:** Hỗ trợ upload tài liệu y khoa.
- **Xử lý tài liệu:** Chia nhỏ file PDF để tối ưu hóa việc xử lý AI.
- **Dịch thuật AI:** Tích hợp Gemini API để thực hiện dịch thuật chuyên sâu.
- **Quản lý hàng đợi (Queue Management):** Xử lý các tác vụ dịch dài bằng hệ thống queue để đảm bảo hiệu suất.
